## Download Instructions

For sites with CAPTCHA protection (RBI, SEBI, CERT-In):

1. Visit the URL in a browser
2. Complete any CAPTCHA/verification
3. Download the PDF document
4. Save to `kb_source_pdfs/` with the filename specified above
5. Convert to Markdown if creating additional KB entries

For sites with direct PDF access:

```bash
# Example — GDPR full text from EUR-Lex
curl -L -o kb_source_pdfs/GDPR_Full_Text_OJ.pdf \
  "https://eur-lex.europa.eu/legal-content/EN/TXT/PDF/?uri=CELEX:32016R0679"

# Example — EU Dual-Use Regulation
curl -L -o kb_source_pdfs/EU_Dual_Use_Regulation_2021.pdf \
  "https://eur-lex.europa.eu/legal-content/EN/TXT/PDF/?uri=CELEX:32021R0821"
```
